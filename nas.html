VM-Based NAS & File Server

A virtualized NAS running on a Windows 11 Pro host to centralize storage for photos, projects, and lab data.

Overview

This NAS lives as a virtual machine on a Windows 11 Pro host. It serves as the main hub for storing photography sets, website assets, project files, and backups. The goal is to get the benefits of a dedicated NAS using existing hardware and virtualization.
